# 3D_Game_of_Life

#### Description
本项目实现了一个 三维生命游戏（3D Game of Life） 的模拟与可视化。生命游戏最初由英国数学家约翰·康威（John Conway）在1970年提出，是一个零玩家的细胞自动机，通过简单的生存和繁衍规则模拟复杂的生命演化过程。相比传统的二维生命游戏，三维版本能够展示更为丰富和复杂的动态行为。
